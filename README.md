# Kaiju Kitty - City Destroyer

A silly browser-based 3D game built with Three.js.

## How to Play
*   **Goal:** Destroy the city! Knock as many buildings off the edge of the world as you can.
*   **Controls:**
    *   **W / Up Arrow:** Move Forward
    *   **S / Down Arrow:** Move Backward
    *   **A / Left Arrow:** Rotate Left
    *   **D / Right Arrow:** Rotate Right

## Technologies
*   [Three.js](https://threejs.org/) - For 3D rendering.
*   HTML5 / CSS3 - For the UI.

## Running Locally
Since this project uses ES modules, you need a local server to run it.

1.  Clone the repository.
2.  Run a local server in the project directory:
    ```bash
    python3 -m http.server
    ```
3.  Open `http://localhost:8000` in your browser.
